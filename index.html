Quick Actions

Item Details

Look up any limited item by its catalog ID. Corresponds to Rolimons.GetItemDetails()

All Items

Browse all non-UGC limited items. Corresponds to Rolimons.GetItems()

Recent Trade Ads

View the latest trade ads from Rolimons. Corresponds to Rolimons.GetRecentAds()

Deal Activity

Raw deal activity data. Corresponds to Rolimons.GetRecentDealActivity()

RAP Calculator

Predict RAP changes when selling at a specific price. Corresponds to Rolimons.CalculateRAPChange()

Input

Uses the same formula from the Lua script: change = floor(|price - RAP| / 10)

Result

Enter values to calculate RAP change.
